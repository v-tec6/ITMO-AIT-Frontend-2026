# Контрамарка — ЛР2

Учебный фронтенд сервиса покупки билетов на мероприятия. Проект работает на чистом HTML, CSS, Bootstrap 5 и JavaScript, а данные берёт из mock API на `json-server`.

## Стек

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Axios
- json-server

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить mock API:

```bash
npm run server
```

3. Открыть фронтенд:

- откройте `index.html` в браузере
- mock API будет доступен по адресу `http://localhost:3000`

## Что реализовано

- загрузка каталога мероприятий с mock API
- единая страница мероприятия `event.html?id=...`
- mock-регистрация и вход через коллекцию `users`
- отображение состояния авторизации в navbar
- покупка билета с созданием заказа в `orders`
- страница `orders.html` с заказами текущего пользователя

## Сценарий демонстрации

1. Откройте `index.html` и покажите каталог мероприятий.
2. Перейдите в карточку любого события.
3. Как гость нажмите `Купить билет` и покажите переход на `login.html`.
4. Войдите под тестовым аккаунтом или зарегистрируйте нового пользователя.
5. Вернитесь на страницу события и оформите покупку билета.
6. Откройте `orders.html` через пункт `Мои билеты` в navbar.
7. Покажите, что новый заказ появился в списке, а количество доступных билетов у события уменьшилось.
