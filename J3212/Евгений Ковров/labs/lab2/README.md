# Контрамарка, ЛР2

Mock API для учебного проекта на `json-server`.

## Запуск

```bash
npm install
npm run server
```

После запуска данные будут доступны по адресу `http://localhost:3000/events`.

Mock-авторизация использует коллекцию `users`, страницы [login.html](D:/Code/frontend/ITMO-AIT-Frontend-2026/J3212/Евгений%20Ковров/labs/lab2/login.html) и [register.html](D:/Code/frontend/ITMO-AIT-Frontend-2026/J3212/Евгений%20Ковров/labs/lab2/register.html), а текущий пользователь хранится в `localStorage`.
